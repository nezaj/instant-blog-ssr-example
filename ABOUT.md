# Instant SSR Demo

A simple microblog demonstrating server-side rendering (SSR) with [InstantDB](https://instantdb.com) and Next.js.

## What's in this repo

- Next.js App Router with SSR
- InstantDB for real-time data
- `useSuspenseQuery` for server-rendered queries
- Magic code authentication with cookie sync
